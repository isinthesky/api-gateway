이 서비스는 API Gateway 테스트용 모의 서비스입니다.

서비스 상태: 온라인

서비스 ID: service1

사용 가능한 엔드포인트

GET /

기본 정보 페이지 (현재 페이지)

GET /api/status

서비스 상태 정보 반환

GET /api/users

사용자 목록 반환

API Gateway를 통해 이 서비스에 액세스하려면 다음 URL을 사용하세요:

http://localhost:8080/ 또는 http://localhost:8080/api/users
