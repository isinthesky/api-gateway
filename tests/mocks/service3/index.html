Service 3 - API Gateway Mock

서비스 정보

이 서비스는 API Gateway 테스트용 모의 서비스입니다.

서비스 상태: 온라인

서비스 ID: service3

사용 가능한 엔드포인트

GET /

기본 정보 페이지 (현재 페이지)

GET /orders

주문 목록 반환

{ "orders": [ {"id": 1, "user_id": 1, "total": 300, "status": "completed"}, {"id": 2, "user_id": 2, "total": 150, "status": "pending"}, {"id": 3, "user_id": 1, "total": 500, "status": "processing"} ] }

GET /orders/:id

특정 주문 정보 반환

{ "id": 1, "user_id": 1, "total": 300, "status": "completed", "items": [ {"product_id": 1, "quantity": 2, "price": 100}, {"product_id": 3, "quantity": 1, "price": 100} ], "shipping_address": "123 Main St, Seoul, Korea", "payment_method": "credit_card", "created_at": "2023-08-01T10:30:00Z" }

POST /orders

새 주문 생성

요청 예시:

{ "user_id": 1, "items": [ {"product_id": 1, "quantity": 2}, {"product_id": 3, "quantity": 1} ], "shipping_address": "123 Main St, Seoul, Korea", "payment_method": "credit_card" }

http://localhost:8080/api/orders
