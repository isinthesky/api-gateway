Service 2 - API Gateway Mock

서비스 정보

이 서비스는 API Gateway 테스트용 모의 서비스입니다.

서비스 상태: 온라인

서비스 ID: service2

사용 가능한 엔드포인트

GET /

기본 정보 페이지 (현재 페이지)

GET /products

제품 목록 반환

{ "products": [ {"id": 1, "name": "Product A", "price": 100}, {"id": 2, "name": "Product B", "price": 200}, {"id": 3, "name": "Product C", "price": 300} ] }

GET /products/:id

특정 제품 정보 반환

{ "id": 1, "name": "Product A", "price": 100, "description": "This is a detailed description of Product A", "stock": 150 }

테스트 지침

API Gateway를 통해 이 서비스에 액세스하려면 다음 URL을 사용하세요:

http://localhost:8080/api/products
